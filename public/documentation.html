myFlixVintage Movie API Documentation

Information

The myFlix API is the server component of a full-stack Java-Script web application that provides access to a vintage-movie database. Built with the MERN stack (MongoDB, Express, React, Node.js), this API stores informations of different movies and users preferences of this movies. The REST API will be accesed using common HTTP methods like GET and POST while (CRUD) will be used to retrieve data from the database and store that data in a non-relational way.

API Endpoints

Movie related API Endpoints

1. Get all the movies

Url: /movies

Method: GET

Request data: None

Response data: A JSON object holding a list of all movies

3. Get genre by description

Url: /genres/:name

Method: GET

Request data: None

Response data: A text message indicating the description of the genre


4. Get infos about a director

Url: /directors/:name

Method: GET

Request data: None

Response data: A JSON object containing name, bio, birth and death-year of a director


5. Get infos about an actor

Url: /actors/:name

Method: GET

Request data: None

Response data: A JSON object containing bio of an actor


User related API Endpoints


6. Register a new user

Url: /users

Method: POST

Request data: A JSON object containing user's details

Response data: A response indicating if the user has been added or not


7. Update user info

Url: /users/:username/:name

Method: PUT

Request data: None

Response data: A response indicating if the user has been added or not


8. Delete user

Url: /users/:username

Method: DELETE

Request data: None

Response data: A response indicating that user has been deleted


9. Add movie to favorites list

Url: /users/:username/favorites/:movieId

Method: POST

Request data: None

Response data: A response that the movie has been added


10. Remove movie from favorites list

Url: /users/:username/favorites/:movieId

Method: DELETE

Request data: None

Response data: A response that the movie has been removed


11. Add movie to To Watch list

Url: /users/:username/towatch/:movieId

Method: POST

Request data: None

Response data: A response that the movie has been added


12. Remove movie from To Watch list

Url: /users/:username/towatch/:movieId

Method: DELETE

Request data: None

Response data: A response that the movie has been removed
