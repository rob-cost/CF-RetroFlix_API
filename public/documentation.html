myFlix API Documentation

Information

The myFlix API is the server component of a full-stack Java-Script web application that provides access to a movie database. Built with the MERN stack (MongoDB, Express, React, Node.js), this API stores informations of different movies and users preferences of this movies. The REST API will be accesed using common HTTP methods like GET and POST while (CRUD) will be used to retrieve data from the database and store that data in a non-relational way.
